La Programmation Web

La programmation web permet selon différentes plates-formes d'édition et l'aide de différents langages de créer un site selon les critères du développeur web. Votre site web peut avoir plusieurs langages notamment celles mentionnées ci-dessous ;

  • REQUIS : HTML5 et CSS3
  • Optionnels : JavaScript, PHP, etc...

La programmation web est désignée à créé des applications webs et des sites web en utilisant l'internet.

Autres informations peuvent être trouvées dans la section « Site d'intérêts », sous le nom « Informations Web # ».
