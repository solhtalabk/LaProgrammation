La Programmation Impérative

La programmation impérative est une façon d'approcher aux programmations (un paradigme), comment une personne fait son code (manière, style).

La programmation impérative suite la séquence d'un code donc il est séquentiel. Cela permet de modifier son état lorsqu'il est instruit de le faire en suivant une suite d'instructions.

On décrit plutôt le comment (le structure du code).

  • Ce paradigme regroupe les programmations de type orienté objet, procédurale et structurée.

Autres informations peuvent être trouvées dans la section « Site d'intérêts », sous le nom « Informations Impérative # ».
