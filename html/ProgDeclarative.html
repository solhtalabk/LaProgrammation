La Programmation Déclarative

La programmation déclarative comparativement aux programmations du type impératives ne suit pas une séquence de codes instruite dans le programme plutôt ça déterminera automatique comment atteindre le résultat souhaité.

Cela fonctionne seulement si les spécifications du résultat sont clairement définies.

On décrit plutôt le quoi (pour résoudre le problème).

  • Ce type de paradigme comporte les langages de programmation logiques et fonctionnelles.

Autres informations peuvent être trouvées dans la section « Site d'intérêts », sous le nom « Informations Déclarative # ».
